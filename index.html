About the repository

The intent of this repo is to:

  1. collect readme files for each of the asset which has been on-boarded to Github Philips-Internal
  2. the readme file corresponding to each asset will have all the metadata about the asset

What are the metadata?

Each AI asset will have following metadata:

  1. Asset Title: (Title of the asset)
  2. Asset Description:
  3. Content Owner:
  4. Asset Category: (Algorithms/Models or Application/Tools)
  5. Location:
  6. Modality: (MR, CT etc)
  7. Technical use case:
  8. Clinical Objective:
  9. Asset type: (Code, SDK, Docker etc)
  10. Cluster: (PD, IGT, PH etc)
  11. AI Lifecycle Phase: (Research, Model creation, Application etc)
  12. Internal Results:
  13. Target BIUs: (ICAP, US, ER PACS etc)
  14. BIU Contact:
  15. External Collaboration:
  16. Data type for evaluation: (MRI-T1, T2, Flair etc)
  17. Type of annotation/labelling for evaluation:
  18. Patents/Publications: (Yes/No)
  19. High Level Approach: (ML/DL/traditional etc)
  20. Release location:
  21. Documentation link:
  22. Source URL:
  23. Innersource URL:

About

files/folder:

  1. assets - contains readme files for each onboarded asset
  2. template_metadata.md - this is the template readme file with all the above mentioned metadata as their fields.
